# GlowAI ✨🧴

GlowAI është një aplikacion web i ndërtuar me React që ndihmon përdoruesit të gjejnë "dupes" (alternativa më të lira) për produktet e tyre të preferuara dhe të shtrenjta të kujdesit të lëkurës (skincare). Ai përdor Inteligjencën Artificiale për të analizuar përbërësit dhe sugjeruar produkte bazuar në cilësinë e formulës.

## 🚀 Funksionalitetet
- Kërkim i bazuar në AI për alternativa të produkteve kozmetike.
- Autentikim i sigurt i përdoruesve (Sign Up / Log In).
- Ruajtja e produkteve të preferuara ("Koleksioni im").
- UI/UX modern, i pastër dhe me menaxhim të qartë të gabimeve.

## 🛠️ Teknologjitë e Përdorura
- **Frontend:** React.js, React Router
- **Backend & Auth:** Supabase
- **AI API:** HuggingFace (për analizën e përbërësve me AI)
- **Deployment:** Vercel
